# BookShelf

My personal reading tracker, and my Angular playground. Add books, mark them read, search your shelf — built step by step while learning Angular the hard (better) way.

## Getting started

\`\`\`bash
npm install
ng serve
\`\`\`

Navigate to `http://localhost:4200/`.

## Learning goals
Practicing Angular fundamentals — components, routing, forms, RxJS, and Signals — through incremental, PR-based development.
